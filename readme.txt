== Description ==

Woo Mpesa vous permet d’intégrer le paiement via mpesa sur un site woocommerce.

Cette extension est une version test et n’intègre pas toutes les fonctionnalités. Vous avez la possibilité d'initier une transaction et de recevoir un mail avec le statut de la transaction.
Avec la version pro, vous pouvez customiser à volonté mais aussi les transactions seront validées automatiquement par votre serveur et le statut woocommerce aussi.

Assurez-vous d'avoir une adresse IP fixe pour votre serveur et enregistrée sur le serveur de MPesa (Vodacom), sinon le votre serveur ne pourra pas envoyer des requêtes vers M-pesa

== Installation ==
Telecharger le plug-in dans le dossier des plugins sur votre FTP
Activer le et aller le configuer dans l'onglet woocommerce > Paiement > Woo Mpesa
